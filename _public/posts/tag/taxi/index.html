Arrival In Buenos Aires
Published on January 6th, 2011
I landed in Buenos Aires yesterday (it seems like a lot longer based on how much I’ve done since then), but so far it is shaping up to be an amazing experience. The flight from Houston was fairly uneventful. I popped a few anti-anxiety pills and then did my best to sleep for as much of the flight as I could. We hit a fairly long bit of turbulence right as we were passing the equator (not sure if that’s the reason), so that kept me from sleeping for a while.
